# Data Privacy & Compliance Prompts

Prompts for GDPR, CCPA/CPRA, data breach response, DPA review, and privacy program compliance.

---

## CCPA/CPRA Compliance Checklist

```
Create a detailed CCPA/CPRA compliance checklist for a [COMPANY TYPE — e.g., U.S.-based
SaaS company / e-commerce retailer] that collects personal information from California
consumers.

Cover:
1. Applicability thresholds and whether the law applies
2. Required privacy notice disclosures
3. Consumer rights: access, deletion, correction, portability, opt-out of sale/sharing
4. Required response procedures and timelines for each request type
5. Sensitive personal information obligations
6. Data minimization and purpose limitation requirements
7. Service provider and contractor agreement requirements
8. Annual cybersecurity audit obligations
9. Employee and contractor training requirements

For each item, include the practical action needed and who is responsible.
```

## GDPR Compliance Checklist

```
Create a detailed GDPR compliance checklist tailored to a [JURISDICTION — e.g., UK-based /
EU-based] [COMPANY TYPE — e.g., SaaS company / data processor].

Cover key obligations under Articles 5–30, including:
1. Core data processing principles (Art. 5)
2. Lawful bases for processing (Art. 6–9)
3. Data subject rights (Art. 15–22)
4. Controller and processor responsibilities (Art. 24–29)
5. Records of processing activities (Art. 30)
6. Data breach notification obligations (Art. 33–34)
7. Data Protection Impact Assessments (Art. 35)
8. International data transfer mechanisms (Ch. V)

For each requirement, include:
- Article reference
- Obligation summary
- Practical action needed
- Evidence or documentation required

Present as a Markdown table suitable for compliance tracking.
```

*Source: The Legal Prompts*

```

team respond to data subject requests under [GDPR / CCPA/CPRA].

Cover the following request types:
- Access / right to know requests
- Deletion / erasure requests
- Correction / rectification requests
- Portability requests
- Opt-out of sale or sharing requests (CCPA)
- Restriction of processing requests (GDPR)

For each:
- Describe the verification steps before responding
- Provide a draft acknowledgment response (to send within required timeframe)
- Provide a draft substantive response template
- Note the applicable response deadline
```

## AI Tool Usage Policy (Internal)

```
Draft a short internal memo or policy outlining rules for employees using AI tools
(e.g., ChatGPT, Claude, Copilot) while protecting:
- Client confidential information
- Personal data subject to GDPR/CCPA
- Trade secrets and proprietary company information
- Attorney-client privileged communications

The policy should:
- Specify what types of information may never be entered into public AI tools
- Describe approved tools and any enterprise agreements in place
- Require review and verification of AI output before use
- Address retention of AI-generated content
- Include a brief training checklist

Tone: Clear and practical. Length: 1–2 pages.
```
